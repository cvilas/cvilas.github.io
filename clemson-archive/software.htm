More Information 		POSIX Utilities Package

A collection of basic C++ utility classes for inter-process message passing, shared memory, synchronization and thread management.


More Information 		QMath - A Matrix Library

QMath is a light-weight matrix library of C++ template classes. The library was developed for robotic applications and is the basis for mathematical operations in the Robotic Platform. Apart from matrix classes, it includes 2nd order filters, integrators, differentiators, routines for homogeneous transformations, and matrix inverse for upto 8x8 matrices.


More Information 		Logitech Wingman USB Joystick Client for QNX

Logitech Wingman Extreme Digital 3D and Force 3D USB joysticks are cheap and compliant with the HID (Human Interface Device) standards. They have eight buttons, a hat switch, a slider and a twist handle providing functionality for a variety of applications. This is a C++ class library for using the joystick under the QNX Real Time Platform Operating System.

Motion Mind DC Motor Controller for UNIX

This is a simple brushed DC motor controller programmable through the serial port. To compile the source code, the library provided by the Posix Utilities Package is required.

More Information 		Mini SSC II Controller in Linux

The Mini SSC II module hooks to the serial port of a PC and can control upto eight pulse proportional servos. This is a C++ software class library for using MiniSSC II in GNU/Linux Operating System.
