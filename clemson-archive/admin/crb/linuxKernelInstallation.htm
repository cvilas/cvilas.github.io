Upgrading the Linux Kernel

This is a quick guide for users who have some experiance working in Linux. Please read the Official Kernel HOWTO for more information. Another excellent resource is this article by Alessandro Rubini.

  1. Download the Linux kernel tarball from http://www.kernel.org
  2. Untar it to the directory /usr/src/
  3. cd /usr/src/linux
  4. make mrproper.

    This will clean up your present configuration file (usually the .config file). If you only want to make minor changes to your present configuration, ignore this step and go to the next one.

  5. make xconfig

    If the last step was ignored, this will result in the configuration currently running on your machine to be loaded. Select your installation componenets. Once the configuration is over, save the configuration as a file in some safe place for future reference.

  6. make dep
  7. make bzImage

    It will take some time. Walk down to the cafe at the corner of the street and get yourself a cup of coffee. Better still, take a long lunch break...

  8. make modules
  9. make modules_install
  10. cp arch/i386/boot/bzImage /boot/LinuxImagexxx
  11. Modify the file /etc/lilo.conf to point to the newly created Linux kernel image.
  12. Run /sbin/lilo
  13. Reboot to run the newly installed kernel.
