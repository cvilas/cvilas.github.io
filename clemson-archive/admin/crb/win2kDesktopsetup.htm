Windows 2000 Installation: The First Steps.
Author: Vilas Kumar Chitrakaran, Aug 2004

  1. It is absolutely essential that you follow these instructions when you setup a new Win2k installation, otherwise you won't last more than 10 minutes on the internet without getting bombed by viruses and worms.
  2. You will need the Win2K installation CD and the Win2K essentials CD. Ask Vilas for it. Make sure you return these CDs back when you are done.
  1. Disconnect the ethernet cable from your PC. You won't need it until the last step in this manual.

  2. Proceed to install Windows 2000 by following on-screen instructions.

  3. After the basic OS installation is complete, configure your network settings as follows
    1. Goto TCP/IP settings (control panel -> network and dialup connections -> local area connection -> properties)
    2. Configure to obtain IP address and DNS settings automatically
    3. Goto My Computer -> properties -> network identification -> properties
    4. Setup your computer name as 'EIBxx-userid', where EIBxx is your room number (example: EIB256) and userid is your DCIT username (example cvilas).
    5. Set workgroup to 'CRB'.


  4. Install Win2K-Service Pack 4 from the Essentials CD.

  5. Install McAfee Antivirus
    1. Install Viruscan from Essentials CD.
    2. Once installation is complete, open up VirusScan Console -> tools -> 'Edit AutoUpdate Repository List'. Move NAIHttp above NAIFtp


  6. Time to connect to the internet. Plug your ethernet cable back in.

  7. Before the urge to check emails or random browsing gets the better of you, do the following:
    1. Goto to windows update and install all critical updates immediately.
    2. Reboot.
    3. Go to windows updates again and make sure you haven't missed any updates.
    4. Update your viruscan to latest virus definition files
    5. Goto control panel->automatic updates and set it up to download and install updates automatically every day.
    6. Goto Viruscan console and set it up to auto update every day as well.


  8. Return the Win2K CDs.
