OCTOR Project Demonstration, DARPA Biodynotics Program
Pennsylvania State University, State College, PA, October 21, 2004

There are images from the first OCTOR demostration to DARPA at the Pennsylvania State University.

The OCTOR soft robotic manipulator arm demonstrated at PSU incorporates twelve air-muscle actuators arranged in four sections, each section consisting of three actuators 120o apart. The whole arm is approximately 1 meter long, with the individual sections capable of both bending (90° - 360°) and extension (~40%). Individual air-muscles are actuated by pressurized air (at up to 90 psi).

The control computer is an Ampro LittleBoard 700 (Pentium III) EBX form-factor Single Board Computer (SBC) running QNX Momentics operating system. The SBC enables untethered remote teleoperation of the robot arm from an Operator Control Unit (OCU) consisting of a laptop with joysticks. The communication link between the OCU and the on-board computer is a point-to-point wireless RF link (IEEE 802.11b). This version of the system is open loop, i.e., no feedback from the robot was utilized by the control system. The control system developed in MATLAB/SIMULINK, and running in the laptop computes air pressures that must to applied to each actuator for the desired robot shape.

The current test platform provides a mission time of about 5 hours when powered by two 12 volts, 33 Ah batteries.
